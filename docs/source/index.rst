Quickstart
==========

1. Install requirements from ``docs/requirements.txt``:

   .. code-block:: bash

      pip install -r docs/requirements.txt

2. Build documentation locally:

   .. code-block:: bash

      cd docs
      make html

3. Open the documentation in a browser at:

   ``docs/build/html/index.html``
